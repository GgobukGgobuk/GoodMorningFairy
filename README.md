# Team: SystemError

## GoodMorningFairy

### Problem Definition

```
일상의 복잡성, 아침에 일어나는 어려움, 수면장애를 개선 한다.
```

### Use Case

```
1. 시간 정보 입력.
2. 정해진 시간에 정해진 위치로 이동.
3. 기상 시간에 알람 울림.
4. 사용자 알람 종료 시 무선 충전으로 이동.
5. 평상시 IR 리모컨 동작.
```
